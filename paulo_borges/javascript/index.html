- Variáveis


alert('Agora é um alerta!')
Comando ALERT é uma ferramenta do Javascript para executar funções através de uma aba.
Comando CONSOLE é uma ferramendo do Javascript para executar funções.

VARIÁVES servem para guardar dados em memória
Existem 3 tipos de variáveis: VAR, LET e CONST
VAR e LET consigo redeclarar, a diferença é só de escopo
CONST não consigo redeclarar
As variáveis servem pra vc não ficar repetindo o mesmo valor muitas vezes.
Uma variável nunca pode iniciar por número, pode iniciar por letra, anderline ou sifrão.
Os númeres podem estar no meio ou no final da variável.
É comum utilizarmos um padrão de escrita camelCase.
